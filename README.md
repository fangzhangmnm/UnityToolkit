# UnityToolkit

Small Unity components I reuse across projects.  
Pick what you need and copy that folder (with `.meta` files).

## What is here

- `BehaviorTree`: SOP-style behavior tree for gameplay AI, with readable syntax sugar.
- `StateMachine`: Three FSM styles (`ClassStateMachine`, `DelegateStateMachine`, `SimpleStateMachine`).
- `ImpactFeel`: Fire-and-forget impact feedback (camera/animation effect helpers).
- `LowRezRendering`: Low-resolution rendering setup, plus editor preview tooling.
- `EditorUtils`: Inspector/editor helpers (`[Button]`, conditional fields, F11 fullscreen, time scale window, oscilloscope helper).
- `PixelFonts`: Lightweight pixel font assets for retro UI.

## Where to start

- If you need AI behavior authoring: open `BehaviorTree/README.md`.
- If you need gameplay state flow: open `StateMachine/README.md`.
- If you need hit/feel polish: open `ImpactFeel/README.md`.
- If you need low-res visuals: open `LowRezRendering/README.md`.
- If you need editor workflow helpers: open `EditorUtils/README.md`.
